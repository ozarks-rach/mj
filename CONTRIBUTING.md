# Contributing to MJ

Thank you for your interest in contributing to the MJ Real Estate Marketplace!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/mj.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Follow the setup instructions in [SETUP.md](./docs/SETUP.md)

## Coding Standards

- Use TypeScript for type safety
- Follow ESLint configuration
- Write meaningful commit messages
- Add tests for new features
- Update documentation as needed

## Pull Request Process

1. Update the README.md with any new features
2. Update relevant documentation
3. Ensure all tests pass
4. Submit your PR with a clear description

## Code Style

- Frontend: Prettier + ESLint
- Backend: ESLint + Prettier
- Database: Follow SQL conventions

## Questions?

Open an issue on GitHub or contact the maintainers.
